# 💳 Smart Banking System (Java)

## 📌 Description
Smart Banking System is a Java-based console application that simulates basic banking operations. It allows users to enter an account number and perform transactions such as deposit, withdrawal, and balance checking. The system also updates the account balance after each transaction.

---

## 🚀 Features
- Enter Account Number
- Deposit Money
- Withdraw Money
- Check Account Balance
- Automatic Balance Update after each transaction

---

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)

---

## 📂 Project Structure
Smart-Banking-System/
│
├── Main.java              # Handles user interaction
├── Account.java           # Base class for account details
├── SavingsAccount.java    # Handles deposit, withdrawal, balance
